v0.2.0 (June 22, 2026)

Enhancements
~~~~~~~~~~~~
* Python version support updated: Py3.10 through Py3.14.  Prior versions may still work but are untested. (:pull:`64`)

* Azimuth=270 logic error fixed with tracking systems (:issue:`61`)
